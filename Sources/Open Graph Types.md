## Open Graph Types

Example syntax:
```
<meta property=“og:type” content=“article” />
```

Common types include
- website
- article
- video
- profile
- description
- locale
- site_name

The types are page specific, not necessarily site specific.

A complete list is at [https://ogp.me/#types](https://ogp.me/#types)
