### Chinese


```dataview
table without id
Quote, Stars as ⭐, file.link as Link from #ideas/proverbs 
where Tradition = "Chinese" AND file.name != "Proverb Note"
sort Stars desc
```


### Western

```dataview
table without id

sort Stars desc
```

### Uncategorized

table without id
